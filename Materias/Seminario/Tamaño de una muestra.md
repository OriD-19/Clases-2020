# Tamaño de una muestra
<div style="text-align: right; opacity: 0.7; font-style: italic;">Creado el 28/07/2022</div>
<div style="text-align: right; opacity: 0.7; font-style: italic;">Última actualización el 28/07/2022</div>

Para calcular el tamaño de una [[Muestra]], se utilizan **fórmulas** para el caso probabilístico.

En el caso de que la población sea mayor a 100,000 personas, entonces se utilizan las siguientes fórmulas:

$$\large n = \frac{Z^2 S^2}{E^2}$$
$$\large n = \frac{Z^2P^2(1-P)}{E^2}$$
